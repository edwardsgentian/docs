# Mintlify Documentation

This is a Mintlify documentation project.

## Getting Started

1. Install the Mintlify CLI:
   ```bash
   npm i -g mint
   ```

2. Start the development server:
   ```bash
   mint dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
